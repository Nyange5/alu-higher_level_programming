1-last_digit.py

Task: Print the last digit of a random number, and add a message depending on the digit.

Hint: Use number % 10 and conditions for >5, 0, <6.

2-print_alphabet.py

Task: Print the lowercase alphabet in one line (no newline after each letter).

Hint: Use a loop and chr() / ord().

3-print_alphabt.py

Task: Print the lowercase alphabet except q and e in one line.

Hint: Similar to above, skip certain letters.

4-print_hexa.py

Task: Print numbers from 0 to 98 in decimal and hexadecimal format.

Hint: Use hex() or format() in the loop.

5-print_comb2.py

Hint: Format numbers with 2 digits, use one loop.

6-print_comb3.py

Hint: Use two loops, skip repeated pairs (01 and 10 are same).
