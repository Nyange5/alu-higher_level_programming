0-add.py

Imports add(a, b) from add_0.py.

Assigns a = 1 and b = 2.

Prints: 1 + 2 = 3 using string formatting.

add_0.py

Defines the function add(a, b) that returns a + b.

Example: add(1, 2) returns 3.


1-calculation.py

Imports all functions from calculator_1.py.

Assigns a = 10, b = 5.

Calls add, sub, mul, div and prints results with formatted strings (max 4 prints).

calculator_1.py

Defines 4 math functions: add, sub, mul, div.


2-args.py

Prints number of command-line arguments and lists them with positions.

Uses sys.argv.

Handles zero or many arguments.


3-infinite_add.py

Adds all command-line arguments as integers.

Prints the sum.

Works with very large numbers.


4-hidden_discovery.py

Prints all names defined in compiled module hidden_4.pyc.

Ignores names starting with __.

Names are printed alphabetically.

Uses import hidden_4 after downloading .pyc from GitHub.

hidden_4.pyc

Compiled Python 3.8 module.

Contains hidden functions/variables like print_hidden, print_school, my_secret_santa.


5-variable_load.py

Contains a single variable: a = 98.

5-variable_load.py imports a and prints its value.
